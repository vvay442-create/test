# AI Dev Timeline

Интерактивная хронология **«Четыре парадигмы AI в разработке за пять лет»** — от
автодополнения строки до автономных агентов. Шаблонный репозиторий для практикума
на митапе.

Стек: чистые HTML + CSS + JS, без сборки. Тесты — vitest + jsdom, линт — ESLint,
деплой — GitHub Pages через GitHub Actions.

## Как получить свою копию

1. Нажми зелёную кнопку **«Use this template» → Create a new repository**.
2. Дай репозиторию имя и создай его под своим аккаунтом.
3. Клонируй копию и поставь зависимости (нужны для тестов и линта):

   ```bash
   npm install
   ```

Сайт можно открыть локально просто двойным кликом по `index.html` — сборка не нужна.

## Как убедиться, что сайт задеплоился

Деплой на GitHub Pages запускается автоматически при пуше в `main`.

1. Открой вкладку **Actions** в своём репозитории и дождись зелёной галочки у
   workflow **Deploy to GitHub Pages**.
2. Сайт будет доступен по адресу:

   ```
   https://<твой-username>.github.io/<имя-репо>/
   ```

3. Ту же ссылку видно в логе job-а деплоя (шаг *deploy*).

> Pages включается автоматически (`enablement: true` в workflow). Если деплой не
> стартовал — проверь Settings → Pages → Source: GitHub Actions.

## Задачи для твоего агента

Поставь агенту одну из задач ниже. **Формулируй критерии приёмки — расплывчатый
запрос даёт расплывчатый PR.**

### 1. Добавить веху

Добавь карточку про **A2A-протокол (апрель 2025)**.
**Критерии приёмки:** данные добавлены в `MILESTONES` в `data.js`; поля
`year/title/description/tag` заполнены; `tag` входит в `PARADIGMS`; `npm test` и
`npm run lint` зелёные.

### 2. Переключатель темы

Кнопка переключения светлой и тёмной темы с сохранением выбора.
**Критерии приёмки:** выбор сохраняется в `localStorage` и применяется при
перезагрузке; есть тест на логику переключения; линт зелёный; новых сетевых
зависимостей нет.

### 3. Анимация появления при скролле

Карточки плавно появляются, когда попадают в область видимости.
**Критерии приёмки:** реализовано через `IntersectionObserver`; работает на
телефоне; уважает `prefers-reduced-motion`; есть тест на навешивание
наблюдателя или класса.

### 4. Фильтр по тегу парадигмы

Панель фильтров: показать только вехи выбранной парадигмы.
**Критерии приёмки:** список фильтров строится из `PARADIGMS`; есть кнопка
«показать все»; есть тест функции фильтрации; линт зелёный.

### 5. Пасхалка по konami code

Ввод ↑ ↑ ↓ ↓ ← → ← → B A запускает скрытый эффект.
**Критерии приёмки:** есть тест на распознавание последовательности; эффект не
ломает страницу; легко отключается.

### 6. Адаптив для узких экранов

Улучши вёрстку для экранов уже 360px.
**Критерии приёмки:** нет горизонтального скролла на 320–375px; читаемые отступы
и типографика; линт зелёный.

## Скрипты

| Команда          | Что делает                              |
| ---------------- | --------------------------------------- |
| `npm test`       | Прогон тестов (vitest + jsdom).         |
| `npm run lint`   | Проверка кода ESLint.                    |

## Структура

```
index.html      разметка страницы
styles.css      тёмная тема и анимация ленты
data.js         MILESTONES + PARADIGMS — единственный источник данных
app.js          рендер карточек хронологии
tests/          тесты vitest (петля верификации для агентов)
AGENTS.md       правила для AI-агентов
```
